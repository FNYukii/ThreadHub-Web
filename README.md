# Thread Hub for Web
## 概要
Webブラウザで利用できる掲示板アプリです。
ユーザーがスレッドを作成し、他のユーザーがそれにコメントを追加していく仕様です。

## 特徴
匿名掲示板となっており、アカウント登録不要で誰でも投稿できます。

## モジュール
- react
- react-router-dom
- react-icons
- dayjs
- firebase
